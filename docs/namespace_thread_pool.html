Detailed Description

This namespace contains the ThreadPool class and related functionality for managing parallel execution of tasks.

FunctionThe type of the functor to apply.
Note
If nitems = 0, it will be computed using std::distance(iter, end).
The redundancy of nitems and iter,end here is due to the fact that, for forward iterators, computing the distance from iterators is costly, and, for input iterators, we might not know in advance how many items there are (e.g., stream iterators).
